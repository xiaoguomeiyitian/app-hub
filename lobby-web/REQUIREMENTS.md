# app-lobby 需求说明

## 定位

`app-lobby` 仅作为应用大厅使用，不承担管理后台、账号体系、游客登录/注册、平台治理、审计或权限中心职责。

## 核心职责

- 展示应用
- 分类导航
- 搜索筛选
- 推荐排序
- 收藏（localStorage）
- 最近玩过（localStorage）
- 轻量分享
- 主题与响应式优化
- 新应用持续接入

## 明确不承担

- 管理后台项目
- 平台控制台
- 账号体系
- 游客注册 / 游客登录
- 管理员权限系统
- 平台治理系统
- 审计平台

## 验收标准

- 首页仅保留应用大厅本体功能
- 无后台入口、无登录/注册入口、无游客入口
- 继续支持应用接入、分类、搜索、收藏、最近使用
- 不再以后台治理为主线扩展
