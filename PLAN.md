# app-hub 项目计划

## 项目目标

构建一个统一的应用大厅平台，集成 104 个前端子项目，提供统一的后端 API、项目发现、路由分发和数据统计能力。

## 目录结构

| 目录 | 用途 |
|------|------|
| `lobby-web/` | 大厅前端（Express 静态服务） |
| `hub-server/` | 共享后端（Express + SQLite） |
| `apps/games/` | 游戏类子项目（33 个） |
| `apps/tools/` | 工具类子项目（47 个） |
| `apps/art/` | 创意艺术类子项目（10 个） |
| `apps/audio/` | 音频类子项目（4 个） |
| `apps/visual/` | 视觉特效类子项目（10 个） |
| `packages/app-utils/` | 前端工具库 |
| `packages/design-system/` | 设计系统 |
| `static/` | 部署产物 |
| `data/` | SQLite 数据库 |
| `docs/` | 技术文档 |
| `scripts/` | 运维脚本 |
| `reports/` | 报告与日志 |
