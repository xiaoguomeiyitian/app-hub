# 二维码生成器

输入文本/URL 生成二维码，自定义颜色，下载 PNG。

```bash
cd /root/projects/apps/qr-generator && npm install && npm run dev
```

## 功能列表
- 文本/URL转二维码
- 自定义二维码样式（颜色、大小、边距）
- 二维码下载（PNG/SVG格式）
- 内置二维码扫描
