# Calculator 使用指南

## 功能说明
Calculator 是一个功能强大的计算器应用，支持基础运算、科学计算、历史记录、主题切换和键盘支持。

### 核心功能
- **基础四则运算**：加、减、乘、除
- **科学计算模式**：支持 sin、cos、tan、log、ln、平方根等
- **历史记录**：自动保存计算历史，可查看和清除
- **主题切换**：支持亮色/暗色模式
- **键盘支持**：支持键盘快捷键操作

## 操作指南

### 基础操作
1. 点击数字按钮或使用键盘输入数字
2. 点击运算符按钮（+、-、×、÷）
3. 点击 "=" 按钮或按 Enter 键计算结果
4. 点击 "C" 按钮或按 Escape 键清除

### 高级功能
- **科学模式**：点击下拉菜单选择 "Scientific" 启用科学计算
- **历史记录**：计算后会自动记录，点击历史项可重新使用
- **主题切换**：点击右上角🌙/☀️按钮切换主题

### 导入/导出
- **导出**：点击右上角 📤 按钮导出当前设置和历史记录
- **导入**：点击 📥 导入按钮选择 JSON 文件导入

### 重置
- 点击右上角 🔄 重置按钮，清除所有数据恢复到默认状态

## 键盘快捷键
- `0-9`：输入数字
- `+ - * /`：运算符（键盘上用 `+ - * /`）
- `Enter`：计算结果
- `Escape`：清除
- `Backspace`：退格
