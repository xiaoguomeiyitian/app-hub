# 正则测试器（Regex Tester）— 需求文档

> 📅 创建日期: 2026-04-02
> 📁 项目目录: `/root/projects/apps/regex-tester/`

## 项目概述

在线正则表达式调试工具，输入正则+文本，实时高亮匹配结果。

## 功能需求

### P0 — 核心

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 正则输入 | 输入正则表达式 | 实时编译 |
| F02 | 修饰符 | g/i/m/s 等开关 | 可勾选 |
| F03 | 测试文本 | 输入待匹配文本 | 多行支持 |
| F04 | 匹配高亮 | 匹配部分在文本中高亮显示 | 每个匹配不同颜色 |
| F05 | 匹配列表 | 列出所有匹配项+索引位置 | 显示完整 |
| F06 | 分组展示 | 捕获分组单独列出 | 编号/命名分组 |
| F07 | 错误提示 | 正则语法错误红色提示 | 定位准确 |

### P1 — 增强

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F08 | 替换模式 | 输入替换字符串，预览替换结果 | 支持 $1 等引用 |
| F09 | 常用模板 | 提供常用正则模板（邮箱/URL/手机等） | 点击填入 |
| F10 | 性能提示 | 正则执行时间+警告回溯 | 显示毫秒数 |
| F11 | 匹配统计 | 匹配数量/耗时 | 实时更新 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 布局 | 上方正则输入+修饰符，中间文本+高亮，下方匹配结果 |
| 风格 | 暗色主题，编辑器风格 |
| 高亮 | 匹配部分交替颜色高亮 |

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端 | 无（纯前端） |
| 部署 | `cp -r dist/* /root/projects/static/regex-tester/` |

## 设备适配

8 分辨率全通过。
