# 需求文档 - ascii-art

> 创建日期: 2026-03-25
> 项目目录: /root/projects/apps/ascii-art

## 功能需求
- 图片转 ASCII 字符画
- 支持字符密度调节
- 支持字符集切换
- 支持彩色模式
- 支持结果复制/下载

## 技术约束
| 项目 | 要求 |
|------|------|
| 前端构建 | Vite + TypeScript (strict) |
| 前端端口 | 20009 |
| 生产构建 | `base: './'` |
| 开发访问 | `server.host: '0.0.0.0'` |

## 验收标准
- [ ] 可通过浏览器访问前端页面
- [ ] 图片转 ASCII 结果正确
