# Calendar 使用指南

## 功能说明
- 月/周/日三种视图切换
- 事件添加、编辑与删除
- 事件提醒设置与重复事件配置
- 事件导入导出（iCal格式）

## 操作指南
### 基础操作
1. 点击顶部"月/周/日"按钮切换日历视图
2. 点击日历空白处，填写事件信息后保存添加事件
3. 点击已有事件查看详情，可进行编辑或删除
4. 拖拽事件可调整时间或日期

## 常见问题
### Q1: 如何设置事件提醒？
A: 添加或编辑事件时，在"提醒"选项中选择提前提醒的时间即可。

### Q2: 支持哪些视图？
A: 支持月视图、周视图、日视图三种显示方式，满足不同查看需求。

### Q3: 事件数据保存在哪里？
A: 默认保存到本地存储，也可导出为iCal文件进行备份。

## 键盘快捷键
| 快捷键 | 功能 |
|--------|------|
| M | 切换到月视图 |
| W | 切换到周视图 |
| D | 切换到日视图 |
| N | 新建事件 |
| E | 编辑选中事件 |
