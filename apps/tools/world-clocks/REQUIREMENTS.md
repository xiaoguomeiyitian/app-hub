# 需求文档 - world-clocks

> 创建日期: 2026-03-25
> 项目目录: /root/projects/apps/world-clocks

## 功能需求
- 全球时钟墙页面
- 同时展示多个城市的时钟
- 支持模拟表盘与昼夜指示
- 支持城市时钟并列展示

| 项目 | 要求 |

| 前端端口 | 20009 |
| 生产构建 | `base: './'` |
| 开发访问 | `server.host: '0.0.0.0'` |
