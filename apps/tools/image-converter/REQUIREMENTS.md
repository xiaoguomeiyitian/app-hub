# 图片工具 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/image-converter/`

## 功能需求

### P0

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 图片上传 | 拖拽/点击上传 | 支持常见格式 |
| F02 | 格式转换 | PNG↔JPG↔WebP↔SVG | 转换正确 |
| F03 | 压缩 | 调节质量(1-100%) | 压缩后可预览 |
| F04 | 尺寸调整 | 指定宽高，支持等比缩放 | 调整正确 |
| F05 | 裁剪 | 可视化裁剪框 | 裁剪正确 |
| F06 | 下载 | 下载处理后图片 | 文件正确 |

### P1

| # | 功能 | 描述 |
|---|------|------|
| F07 | Base64 互转 | 图片↔Base64 |
| F08 | 批量处理 | 多张图片批量转换 |
| F09 | EXIF 查看 | 查看图片元信息 |

## 设备适配

- [ ] 375~1920 全部无溢出
- [ ] 拖拽上传在触控设备可用
- [ ] 裁剪框可触控调整
