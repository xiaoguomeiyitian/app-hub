# 时间戳转换 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/timestamp-tool/`

## 项目概述

提供时间戳与日期之间的双向转换，支持按时区解释与批量查询，面向日常开发与排障场景。

## 功能需求

### P0

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 时间戳转日期 | 支持秒 / 毫秒 / 微秒 | 转换正确 |
| F02 | 日期转时间戳 | 本地日期时间输入 | 秒 / 毫秒 / 微秒输出正确 |
| F03 | 当前时间填充 | 一键填入当前时间 | 可用 |
| F04 | 实时更新 | 当前时间卡片实时刷新 | 每秒更新 |
| F05 | 复制结果 | 复制转换结果 | 可用 |
| F06 | 多时区预览 | 预览多个时区的对应时间 | 可用 |
| F07 | 时区搜索 | 可通过城市/时区名搜索 | 可用 |
| F08 | 批量转换 | 多行时间戳批量转换 | 可用 |

### P1

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F09 | UTC 显示 | 显示 UTC / ISO | 正确 |
| F10 | 响应式 | 手机到桌面 | 无溢出 |
