# Color Picker 使用指南

## 功能说明
- 颜色拾取（屏幕取色/输入色值）
- 颜色格式转换（HEX/RGB/HSL/CMYK）
- 配色方案生成
- 历史颜色记录

## 操作指南
### 基础操作
1. 点击"拾色器"按钮，在屏幕上选取颜色
2. 直接在输入框输入HEX/RGB等格式的色值
3. 点击对应格式的"复制"按钮复制到剪贴板

### 高级功能
- 格式转换：自动转换HEX、RGB、HSL、CMYK四种格式
- 配色方案：根据选中颜色生成互补色、邻近色等配色方案
- 历史记录：保存最近拾取的颜色，方便回溯使用

## 常见问题
### Q1: 如何拾取屏幕上的颜色？
A: 点击"屏幕取色"按钮，鼠标变成取色器，点击目标颜色即可获取。

### Q2: 支持哪些颜色格式？
A: 支持HEX、RGB、HSL、CMYK四种常用格式，自动实时转换。

### Q3: 配色方案怎么用？
A: 选中主色后，工具会自动生成多种配色方案，点击可复制对应色值。

## 键盘快捷键
| 快捷键 | 功能 |
|--------|------|
| P | 打开屏幕取色器 |
| C | 复制当前选中色值 |
| H | 查看历史颜色 |
