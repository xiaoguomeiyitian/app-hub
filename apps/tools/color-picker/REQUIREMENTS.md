# 取色器 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/color-picker/`
> ⚠️ 替代/增强旧 color-converter（仅 16K 静态）

## 项目概述

可视化取色工具，支持多种取色方式和格式转换。

## 功能需求

### P0

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 色轮取色 | 圆形色轮 + 明度滑块 | 取色准确 |
| F02 | 滑块取色 | R/G/B/H/S/L 滑块 | 实时更新 |
| F03 | 输入取色 | 直接输入 HEX/RGB/HSL 值 | 解析正确 |
| F04 | 格式转换 | HEX ↔ RGB ↔ HSL ↔ CMYK | 转换准确 |
| F05 | 一键复制 | 点击复制色值 | 复制成功 |

### P1

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F06 | 色板收藏 | 收藏常用颜色 | localStorage |
| F07 | 对比度检查 | WCAG AA/AAA 对比度 | 计算正确 |
| F08 | 调色板生成 | 基于主色生成互补/类似/三色 | 生成合理 |

## 设备适配

- [ ] 375~1920 全部无溢出
- [ ] 色轮在手机端可触控拖拽
- [ ] 滑块触控可用
