# Notepad 使用指南

## 功能说明
- 实时文本编辑，自动保存到本地存储
- 支持Markdown语法编辑与实时预览
- 多笔记管理，支持搜索与分类
- 导出为TXT/MD格式

## 操作指南
### 基础操作
1. 点击左上角"新建笔记"创建新笔记
2. 在编辑区输入文本内容，自动保存无需手动操作
3. 点击笔记列表中的笔记切换编辑不同笔记
4. 点击"删除"按钮删除选中笔记

### 高级功能
- Markdown预览：点击顶部"预览"按钮切换编辑/预览模式，实时渲染Markdown效果
- 笔记搜索：使用顶部搜索栏输入关键词，快速筛选匹配的笔记
- 导出笔记：点击笔记右上角"导出"按钮，选择TXT或MD格式下载

## 常见问题
### Q1: 笔记会自动保存吗？
A: 是的，所有编辑内容会自动保存到浏览器本地存储，刷新页面或关闭浏览器后数据不会丢失。

### Q2: 支持哪些Markdown语法？
A: 支持标题、加粗、斜体、列表、链接、代码块、引用等常用Markdown语法。

### Q3: 如何导出笔记？
A: 打开需要导出的笔记，点击右上角"导出"按钮，选择导出格式（TXT/MD）即可下载到本地。

## 键盘快捷键
| 快捷键 | 功能 |
|--------|------|
| Ctrl+S | 手动保存（补充自动保存机制） |
| Ctrl+B | 加粗选中文本 |
| Ctrl+I | 斜体选中文本 |
| Ctrl+P | 切换Markdown预览模式 |
| Ctrl+N | 新建笔记 |
