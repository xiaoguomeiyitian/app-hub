# Password Generator 使用指南

## 功能说明
- 随机密码生成（自定义长度、字符类型）
- 密码强度检测
- 批量生成密码
- 排除易混淆字符

## 操作指南
### 基础操作
1. 设置密码长度（滑动条或输入框）
2. 选择包含的字符类型（大写、小写、数字、符号）
3. 点击"生成"按钮生成随机密码
4. 点击"复制"按钮复制到剪贴板

### 高级功能
- 密码强度检测：实时显示生成密码的强度等级（弱/中/强）
- 排除相似字符：勾选后排除0/O、1/l等易混淆字符
- 批量生成：点击"批量生成"一次性生成多个密码供选择

## 常见问题
### Q1: 生成的密码安全吗？
A: 采用加密安全随机数生成，确保密码随机性和唯一性，符合安全规范。

### Q2: 如何复制密码？
A: 点击密码旁的"复制"按钮，即可复制到剪贴板，直接使用。

### Q3: 支持哪些字符类型？
A: 支持大写字母、小写字母、数字、特殊符号四种类型，可自由组合勾选。
