# Kanban 使用指南

## 功能说明
- 可视化看板管理（待办/进行中/已完成三列）
- 卡片拖拽排序与列间移动
- 任务优先级标记与截止日期设置
- 任务过滤与搜索

## 操作指南
### 基础操作
1. 点击"+"按钮在对应列添加新任务卡片
2. 拖拽卡片可调整顺序或移动到其他列
3. 点击卡片内容进入编辑模式，修改任务信息
4. 点击卡片右上角"删除"按钮删除任务

### 高级功能
- 优先级设置：为任务标记高/中/低优先级，用不同颜色区分
- 截止日期：为任务设置截止日期，到期自动提醒
- 过滤任务：使用顶部过滤栏按优先级或状态筛选任务

## 常见问题
### Q1: 卡片可以拖拽吗？
A: 是的，支持拖拽卡片调整顺序，或在不同列之间移动任务。

### Q2: 如何删除任务？
A: 点击任务卡片右上角的"删除"按钮，确认后即可删除该任务。

### Q3: 数据会保存吗？
A: 所有看板数据自动保存到本地存储，刷新页面不会丢失数据。

## 键盘快捷键
| 快捷键 | 功能 |
|--------|------|
| N | 新建任务 |
| E | 编辑选中任务 |
| D | 删除选中任务 |
| F | 打开过滤面板 |
