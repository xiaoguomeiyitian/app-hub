# kanban

看板任务管理工具。

## 功能
- 当前项目以现有代码实现为准，提供对应的核心交互或游戏玩法。
- 界面与交互已集成在前端主入口中。

## 运行

```bash
cd /root/projects/apps/kanban && npm install && npm run dev
```

## 构建

```bash
cd /root/projects/apps/kanban && npm run build
```

## 说明
- 前端采用 Vite + TypeScript。
- 生产构建使用相对路径 `base: './'`。
- 若项目包含联机能力，请以代码中的 WebSocket 路径与共享后端路由为准。

## 功能列表
- 可视化看板管理（待办/进行中/已完成）
- 卡片拖拽排序与移动
- 任务优先级标记
- 任务截止日期设置
