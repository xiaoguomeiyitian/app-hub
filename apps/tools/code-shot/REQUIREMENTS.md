# 代码截图（Code Shot）— 需求文档

> 📅 创建日期: 2026-04-02
> 📁 项目目录: `/root/projects/apps/code-shot/`

## 项目概述

将代码片段美化为高质量截图，支持多种主题和语法高亮。

## 功能需求

### P0 — 核心功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 代码输入 | 多行文本编辑器 | 支持粘贴/输入 |
| F02 | 语言选择 | 下拉选择编程语言 | 至少支持 10 种语言 |
| F03 | 语法高亮 | 按语言规则着色 | 高亮正确 |
| F04 | 主题切换 | 多种配色主题（Dracula/Monokai/Nord/GitHub等） | ≥5 种主题 |
| F05 | 截图预览 | 实时预览美化后的代码截图 | 渲染正确 |
| F06 | 导出 PNG | 点击下载为 PNG 图片 | 图片质量高，背景透明可选 |
| F07 | 窗口样式 | 模拟 macOS 窗口（三色按钮） | 可开关 |
| F08 | 行号显示 | 代码左侧显示行号 | 可开关 |

### P1 — 增强

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F09 | 字体选择 | 等宽字体选择 | ≥3 种 |
| F10 | 字号调整 | 代码字号滑块 | 范围合理 |
| F11 | 内边距调整 | 截图四周留白 | 滑块控制 |
| F12 | 水印 | 可选水印文字 | 可开关 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 布局 | 左侧编辑器，右侧实时预览 |
| 风格 | 暗色主题为主，简洁现代 |
| 编辑器 | 等宽字体，带行号 |
| 预览 | 类 Carbon.dev 风格 |

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端 | 无（纯前端） |
| 截图 | html2canvas 或 Canvas 绘制 |
| 部署 | `cp -r dist/* /root/projects/static/code-shot/` |

## 设备适配

8 分辨率全通过，小屏上下堆叠布局。
