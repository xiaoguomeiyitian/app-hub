# Pomodoro 使用指南

## 功能说明
- 标准番茄钟计时（25分钟工作+5分钟休息）
- 自定义计时时长
- 任务关联与完成统计
- 计时结束声音与通知提醒

## 操作指南
### 基础操作
1. 点击"开始"按钮启动番茄钟计时
2. 计时过程中可点击"暂停"暂停计时，点击"继续"恢复
3. 点击"重置"按钮恢复初始计时状态
4. 选择当前任务，完成后自动标记番茄钟完成

### 高级功能
- 自定义时长：点击右上角"设置"图标，修改工作/休息时长
- 任务绑定：下拉选择当前进行的任务，番茄钟完成后自动更新任务进度
- 统计查看：点击"统计"按钮查看今日/本周完成的番茄钟数量

## 常见问题
### Q1: 计时结束后有提示吗？
A: 是的，计时结束会播放提示音并显示浏览器通知，确保你及时知晓。

### Q2: 如何修改计时时长？
A: 点击右上角"设置"图标，在设置面板中调整工作时长和休息时长，支持自定义。

### Q3: 统计数据会保存吗？
A: 所有统计数据自动保存到本地存储，清除浏览器数据会丢失历史记录。

## 键盘快捷键
| 快捷键 | 功能 |
|--------|------|
| Space | 开始/暂停计时 |
| R | 重置计时 |
| S | 打开设置面板 |
| T | 查看统计面板 |
