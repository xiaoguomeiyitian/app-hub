# 番茄钟（Pomodoro）— 需求文档

> 📅 创建日期: 2026-04-02
> 📁 项目目录: `/root/projects/apps/pomodoro/`

## 项目概述

简洁番茄钟专注计时器，25/5/15 三段循环。

> 注意：本项目是简化版番茄钟。`pomodoro-pro` 为增强版（含任务管理/历史统计），两者定位不同。

## 功能需求

### P0 — 核心

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 计时器 | 25 分钟工作 / 5 分钟短休 / 15 分钟长休 | 倒计时准确 |
| F02 | 开始/暂停 | 点击开始，再次点击暂停 | 状态切换正确 |
| F03 | 重置 | 重置当前阶段计时 | 时间归零 |
| F04 | 阶段切换 | 工作→短休→工作→...→长休（每 4 轮） | 循环正确 |
| F05 | 自动开始 | 阶段结束自动进入下一阶段 | 可配置 |
| F06 | 完成计数 | 统计已完成的番茄数 | 持久化 localStorage |
| F07 | 浏览器通知 | 阶段结束时弹通知 | 需用户授权 |

### P1 — 增强

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F08 | 自定义时长 | 可调整工作/短休/长休分钟数 | 范围合理 |
| F09 | 进度环 | 圆形进度条显示剩余时间 | 动画流畅 |
| F10 | 今日统计 | 今天完成的番茄数 | 跨日重置 |
| F11 | 音效 | 阶段结束提示音 | 可开关 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 风格 | 极简居中，大面积留白 |
| 计时器 | 大号数字，圆形进度环 |
| 颜色 | 工作=红色调，休息=绿色调 |
| 按钮 | 开始/暂停/重置 三个按钮 |

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端 | 无（纯前端） |
| 部署 | `cp -r dist/* /root/projects/static/pomodoro/` |

8 分辨率全通过。
