# CSS 沙盒 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/css-playground/`

## 项目概述

实时 CSS 编辑与预览沙盒，开发者可在左侧面板编写 CSS，右侧实时看到效果。

## 功能需求

### P0

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | HTML 面板 | 左侧编写 HTML | 语法高亮 |
| F02 | CSS 面板 | 左侧编写 CSS | 语法高亮 |
| F03 | 实时预览 | 右侧 iframe 实时渲染 | 编辑即更新，无延迟感 |
| F04 | 响应式预览切换 | 切换手机/平板/桌面预览尺寸 | 尺寸准确 |
| F05 | 导出代码 | 导出 HTML/CSS 文件或复制 | 导出正确 |

### P1

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F06 | 示例模板 | 内置常用 CSS 效果模板 | 可加载 |
| F07 | 分屏布局 | 左右/上下分屏切换 | 布局正确 |
| F08 | 本地保存 | localStorage 保存代码 | 刷新不丢失 |
| F09 | 全屏预览 | 预览区全屏 | 正常 |

## 设备适配

- [ ] 375~1920 全部无溢出
- [ ] 移动端上下分屏
- [ ] 代码编辑器触控滚动
