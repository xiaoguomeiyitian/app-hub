# 潜行快递 — 联机版需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/stealth-mp/`
> 关联: `/root/projects/apps/stealth-express/`（已有完整单机版 1947 行）

## 项目概述

在现有潜行快递单机版基础上，增加非对称多人对抗模式。

## 功能需求

### P0 — 联机功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 非对称对抗 | 1 名快递员 vs 1-3 名保安 | 阵营分配正确 |
| F02 | 快递员目标 | 搜集货物→到达撤离点 | 目标明确 |
| F03 | 保安目标 | 巡逻+发现+追击快递员 | 保安有策略性 |
| F04 | 视野系统 | 快递员/保安各有视野范围 | 迷雾正确 |
| F05 | 地图共享 | 同一地牢地图 | 地图一致 |
| F06 | 实时同步 | WebSocket 同步移动+交互 | 延迟 < 100ms |
| F07 | 好友房 | 房间码 | 同局 |
| F08 | BOT 补位 | 缺人时 AI 接管 | AI 行为合理 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 风格 | 复用单机版风格 |
| 视野 | 快递员视野小，保安视野大 |
| HUD | 阵营标识+任务提示 |

## 设备适配

**适配验收清单**：
- [ ] 375~1920 全部无溢出
- [ ] 移动端虚拟摇杆可用
- [ ] 横屏优先

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | 复用 stealth-express 源码，增加联机模块 |
| 后端路由 | `shared-backend/src/routes/stealth-mp/` |
| WebSocket | 原生 `ws` `noServer` 模式 |
| 部署 | `cp -r dist/* /root/projects/static/stealth-express/`（覆盖原部署） |
