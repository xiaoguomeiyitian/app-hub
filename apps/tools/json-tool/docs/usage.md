# JSON Tool 使用指南

## 功能说明
- JSON格式化与压缩
- JSON语法校验与错误定位
- JSON转CSV/XML/YAML格式
- JSON对比功能

## 操作指南
### 基础操作
1. 在左侧输入框粘贴或输入JSON内容
2. 点击"格式化"按钮美化JSON结构，添加缩进和换行
3. 点击"压缩"按钮去除空格和换行，压缩JSON体积
4. 点击"校验"按钮检查JSON语法是否正确

### 高级功能
- 语法校验：自动检测JSON语法错误，点击错误提示定位具体位置
- 格式转换：格式化后点击"转换"按钮选择目标格式（CSV/XML/YAML）
- 对比JSON：粘贴两个JSON文件，点击"对比"按钮查看差异

## 常见问题
### Q1: JSON格式错误怎么办？
A: 工具会自动检测错误，点击错误提示可定位到具体行数和位置。

### Q2: 支持大文件吗？
A: 支持最大10MB的JSON文件处理，过大的文件可能出现卡顿。

### Q3: 如何转换格式？
A: 格式化JSON后，点击"转换"按钮选择目标格式即可下载转换后的文件。

## 键盘快捷键
| 快捷键 | 功能 |
|--------|------|
| Ctrl+Shift+F | 格式化JSON |
| Ctrl+Shift+M | 压缩JSON |
| Ctrl+E | 校验语法 |
| Ctrl+D | 对比JSON |
