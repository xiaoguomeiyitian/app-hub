# JSON 格式化（JSON Tool）— 需求文档

> 📅 创建日期: 2026-04-02
> 📁 项目目录: `/root/projects/apps/json-tool/`

## 项目概述

JSON 查看、格式化、压缩、校验工具。

## 功能需求

### P0 — 核心

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | JSON 输入 | 大文本编辑器 | 支持粘贴 |
| F02 | 格式化 | 缩进美化（2/4 空格可选） | 输出正确 |
| F03 | 压缩 | 去除所有空白 | 输出正确 |
| F04 | 校验 | 检查 JSON 是否合法 | 错误定位到行列 |
| F05 | 错误提示 | 语法错误红色标记+位置提示 | 定位准确 |
| F06 | 树状视图 | 可折叠的 JSON 树形展开 | 层级缩进正确 |
| F07 | 一键复制 | 复制格式化结果到剪贴板 | 复制成功 |

### P1 — 增强

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F08 | JSONPath 查询 | 输入路径过滤数据 | 基础路径查询 |
| F09 | 转义/反转义 | JSON 字符串转义处理 | 正确 |
| F10 | 对比模式 | 两份 JSON 并排对比 | 差异高亮 |
| F11 | 统计信息 | 键值对数量、深度、大小 | 显示正确 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 布局 | 上方输入，下方格式化结果/树状视图切换 |
| 风格 | 暗色编辑器风格 |
| 树状视图 | 彩色语法高亮，可折叠 |

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端 | 无（纯前端） |
| 部署 | `cp -r dist/* /root/projects/static/json-tool/` |
