# 流程图编辑器（Flow Editor）— 需求文档

> 📅 创建日期: 2026-04-02
> 📁 项目目录: `/root/projects/apps/flow-editor/`

## 项目概述

拖拽式流程图编辑器，支持节点创建、连线、编辑和导出。

## 功能需求

### P0 — 核心

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 画布 | 无限滚动画布 | 拖拽平移 |
| F02 | 节点类型 | 矩形/圆角矩形/菱形/圆形 | 4 种基本形状 |
| F03 | 添加节点 | 从侧栏拖拽到画布 | 创建成功 |
| F04 | 节点文字 | 双击节点编辑文字 | 支持多行 |
| F05 | 连线 | 从节点锚点拖拽创建连线 | 贝塞尔曲线连接 |
| F06 | 选择/移动 | 点击选中，拖拽移动 | 多选支持 |
| F07 | 删除 | Delete 键删除选中 | 节点和连线均可删 |
| F08 | 缩放 | 滚轮缩放画布 | 范围 25%-400% |

### P1 — 增强

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F09 | 撤销/重做 | Ctrl+Z / Ctrl+Y | 操作历史完整 |
| F10 | 连线箭头 | 线条末端箭头 | 方向正确 |
| F11 | 节点颜色 | 修改填充/边框颜色 | 调色板选择 |
| F12 | 导出 SVG | 下载为 SVG 矢量图 | 可打开 |
| F13 | 导出 PNG | 下载为 PNG 图片 | 分辨率清晰 |
| F14 | 本地保存 | localStorage 保存当前图 | 刷新不丢失 |
| F15 | 对齐网格 | 拖拽时吸附网格 | 可开关 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 布局 | 左侧节点面板，中央画布，顶部工具栏 |
| 风格 | 浅色画布，简洁节点样式 |
| 节点 | 白色填充 + 灰色边框，双击编辑 |
| 连线 | 灰色贝塞尔曲线 + 箭头 |

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端 | 无（纯前端） |
| 渲染 | SVG（适合矢量图导出） |
| 部署 | `cp -r dist/* /root/projects/static/flow-editor/` |

## 设备适配

8 分辨率全通过，触屏支持拖拽操作。
