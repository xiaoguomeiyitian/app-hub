# Markdown 编辑器（MD Editor）— 需求文档

> 📅 创建日期: 2026-04-02
> 📁 项目目录: `/root/projects/apps/md-editor/`

## 项目概述

左右分栏 Markdown 编辑器，左侧编辑右侧实时预览。

## 功能需求

### P0 — 核心

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 编辑器 | 左侧 Markdown 编辑区 | 支持多行编辑 |
| F02 | 实时预览 | 右侧 HTML 渲染 | 更新延迟 < 100ms |
| F03 | 基础语法 | 标题/加粗/斜体/删除线/链接/图片/代码块 | 渲染正确 |
| F04 | 列表 | 有序/无序列表/嵌套 | 渲染正确 |
| F05 | 表格 | Markdown 表格渲染 | 对齐正确 |
| F06 | 引用 | 块引用渲染 | 样式正确 |
| F07 | 水平线 | `---` 分割线 | 渲染正确 |
| F08 | 代码高亮 | 代码块内语法高亮 | 支持常见语言 |

### P1 — 增强

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F09 | 工具栏 | 常用格式按钮（B/I/链接/图片/代码） | 点击插入语法 |
| F10 | 同步滚动 | 编辑区和预览区同步滚动 | 滚动对齐 |
| F11 | 导出 HTML | 下载为 HTML 文件 | 格式完整 |
| F12 | 导出 MD | 下载为 .md 文件 | 内容正确 |
| F13 | 字数统计 | 实时字数/行数 | 显示准确 |
| F14 | 全屏预览 | 隐藏编辑区，全屏看预览 | 切换流畅 |
| F15 | 本地保存 | localStorage 自动保存 | 刷新不丢失 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 布局 | 左右等宽分栏，可拖拽调整比例 |
| 编辑器 | 等宽字体，暗色背景 |
| 预览 | 白色背景，排版美观 |
| 工具栏 | 顶部固定，图标按钮 |

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端 | 无（纯前端） |
| Markdown 解析 | 轻量自写或 marked 库 |
| 部署 | `cp -r dist/* /root/projects/static/md-editor/` |

## 设备适配

8 分辨率全通过，小屏上下堆叠（编辑在上，预览在下）。
