# Todo App 使用指南

## 功能说明
- 任务添加、编辑、删除
- 任务分类与过滤
- 任务完成统计
- 本地存储自动保存

## 操作指南
### 基础操作
1. 在输入框输入任务内容，点击"添加"按钮添加任务
2. 点击任务前的复选框标记任务完成
3. 点击任务后的"删除"按钮删除任务
4. 点击任务内容进入编辑模式，修改后保存

### 高级功能
- 任务分类：为任务设置工作/生活/学习等分类标签
- 过滤任务：按状态（全部/未完成/已完成）或分类过滤显示
- 统计查看：查看今日/本周完成的任务数量和进度

## 常见问题
### Q1: 任务会保存吗？
A: 所有任务自动保存到本地存储，刷新页面不会丢失数据。

### Q2: 如何编辑任务？
A: 点击任务内容，进入编辑模式，修改后点击"保存"即可。

### Q3: 支持哪些过滤方式？
A: 支持按完成状态（全部/未完成/已完成）和任务分类过滤。

## 键盘快捷键
| 快捷键 | 功能 |
|--------|------|
| Enter | 添加新任务 |
| E | 编辑选中任务 |
| D | 删除选中任务 |
| F | 打开过滤面板 |
