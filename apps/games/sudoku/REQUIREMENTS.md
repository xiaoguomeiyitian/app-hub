# 数独（Sudoku）— 需求文档

> 📅 创建日期: 2026-04-02
> 📁 项目目录: `/root/projects/apps/sudoku/`

## 项目概述

经典数独游戏，支持**单人模式** + **多人竞速对战模式**。

## 功能需求

### P0 — 单人模式

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 9×9 棋盘 | 标准数独网格，3×3 宫格高亮 | 渲染正确 |
| F02 | 预填数字 | 题目给出部分数字（不可修改） | 预填格不可编辑 |
| F03 | 填数 | 点击空格 → 输入 1-9 | 填入正确 |
| F04 | 橡皮擦 | 清除已填数字 | 清除正确 |
| F05 | 冲突高亮 | 同行/列/宫重复数字标红 | 实时检测 |
| F06 | 胜利判定 | 所有空格填对即胜 | 判定正确 |
| F07 | 计时器 | 从开始到完成的用时 | 精度秒 |
| F08 | 难度选择 | 简单/中等/困难（空格数递增） | 3 档难度 |
| F09 | 笔记模式 | 空格可标记候选数字 | 候选数显示 |
| F10 | 撤销/重做 | 支持多步撤销和重做 | 操作历史正确 |

### P1 — 联机竞速

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F11 | 快速匹配 | 2 人同题竞速 | 匹配成功 |
| F12 | 创建房间 | 房间码分享 | 加入成功 |
| F13 | 实时进度 | 显示对手完成格数 | 同步正确 |
| F14 | 竞速胜负 | 先完成且正确者胜 | 判定正确 |
| F15 | WebSocket 心跳 | 20 秒 ping/pong | 不断线 |

### P2 — 体验增强

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F16 | 数字键盘 | 1-9 虚拟键盘 | 触屏可操作 |
| F17 | 高亮同数 | 选中数字高亮所有相同数字 | 便于观察 |
| F18 | 最佳记录 | localStorage 存最佳时间 | 按难度分档 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 整体风格 | 白底浅灰网格，简洁清爽 |
| 宫格边框 | 3×3 宫格加粗边框 |
| 选中高亮 | 选中格+同行/列/宫浅色背景 |
| 冲突标记 | 红色数字+红框 |
| 预填数字 | 加粗黑色 |
| 手填数字 | 普通蓝色 |

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端 | 已有 `shared-backend/src/routes/sudoku/index.ts` |
| WebSocket | `/api/sudoku/websocket` |
| 后端协议 | `match:join`/`room:create`/`room:join` → `game:start` → `game:fill` → `game:over` |
| 部署 | `cp -r dist/* /root/projects/static/sudoku/` |

## 设备适配

8 分辨率全通过，手机端数字键盘操作友好。
