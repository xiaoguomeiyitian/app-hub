# 跳棋 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/checkers/`

## 项目概述

跳棋游戏，支持 AI + 在线多人。

## 功能需求

### P0

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 棋盘 | 8×8 黑白格 | 渲染正确 |
| F02 | 棋子 | 红/黑各 12 枚 | 初始摆位正确 |
| F03 | 移动 | 斜走一格 | 规则正确 |
| F04 | 跳吃 | 跳过对方棋子吃掉 | 连跳支持 |
| F05 | 升王 | 到底线变王，可后退 | 升王正确 |
| F06 | 胜负 | 无子可动或无子→输 | 判定正确 |
| F07 | AI | Minimax | 有挑战性 |

### P1 — 联机

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F08 | 2/4/6 人 | 多人跳棋 | 同步正确 |
| F09 | 好友房 | 房间码 | 同局 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 风格 | 经典黑白格，彩色棋子 |
| 移动 | 滑动动画 |
| 升王 | 皇冠特效 |

## 设备适配

**适配验收清单**：
- [ ] 375~1920 全部无溢出
- [ ] 棋子可点击/拖拽（触控）
- [ ] 棋盘完整可见

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端路由 | `shared-backend/src/routes/checkers/` |
| WebSocket | 原生 `ws` `noServer` 模式 |
| 部署 | `cp -r dist/* /root/projects/static/checkers/` |
