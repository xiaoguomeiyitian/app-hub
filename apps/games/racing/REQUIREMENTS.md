# 像素赛车 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/racing/`

## 项目概述

像素风格赛车游戏，支持单机 + 在线竞速。

## 功能需求

### P0

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 赛道 | 俯视角/侧视角，随机生成 | 生成合理 |
| F02 | 车辆控制 | 加速/刹车/左右转向 | 操控流畅 |
| F03 | 计时赛 | 单圈计时 | 计时精确 |
| F04 | 闯关赛 | 多关卡递增难度 | 关卡递进 |
| F05 | 车辆升级 | 速度/加速度/操控 | 升级生效 |
| F06 | 氮气加速 | 收集氮气瓶→加速 | 加速效果 |

### P1 — 联机

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F07 | 2-4 人竞速 | 同赛道比快 | 位置同步 |
| F08 | 幽灵赛 | 回放对手跑法 | 与幽灵同赛道 |
| F09 | 好友房 | 房间码 | 同局 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 风格 | 像素风俯视角赛道 |
| 特效 | 氮气火焰、漂移火花 |
| HUD | 速度表/排名/计时 |

## 设备适配

**适配验收清单**：
- [ ] 375~1920 全部无溢出
- [ ] 移动端虚拟方向盘+油门刹车
- [ ] 横屏优先提示
- [ ] 桌面端方向键/WASD

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端路由 | `shared-backend/src/routes/racing/` |
| WebSocket | 原生 `ws` `noServer` 模式 |
| 部署 | `cp -r dist/* /root/projects/static/racing/` |
