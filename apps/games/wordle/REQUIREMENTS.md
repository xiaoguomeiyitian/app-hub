# Wordle 猜词 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/wordle/`

## 项目概述

Wordle 猜词游戏，支持单机 + 在线竞速对战。

## 功能需求

### P0 — 核心功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 猜词机制 | 5 字母单词，6 次机会 | 猜中/用完结束 |
| F02 | 颜色反馈 | 绿(位置对)/黄(有这字母位置错)/灰(无这字母) | 反馈正确 |
| F03 | 虚拟键盘 | 屏幕键盘显示字母状态 | 状态实时更新 |
| F04 | 每日一词 | 每日固定一词，全球一致 | 同一天所有人同一词 |
| F05 | 无限模式 | 随机词无限玩 | 每局新词 |
| F06 | 词库验证 | 输入必须是合法单词 | 非词库词拒绝 |
| F07 | 难度选择 | 4/5/6/7/8 字母可选 | 难度切换 |

### P1 — 联机功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F08 | 1v1 竞速 | 同词比谁先猜中 | 实时显示对方进度 |
| F09 | 好友房 | 房间码 | 2 人同场 |
| F10 | 排行榜 | 每日排名 | SQLite |

## UI 要求

| 项目 | 要求 |
|------|------|
| 风格 | 简洁白色背景，绿色主题色 |
| 网格 | 5×6（或对应难度）方格 |
| 翻转动画 | 猜词后每个格子翻转显示颜色 |
| 键盘 | 三行 QWERTY 布局，颜色同步 |

## 设备适配

**适配验收清单**：
- [ ] 375/390/412/768/1024/1280/1366/1920 全部无溢出
- [ ] 虚拟键盘在手机端占合适比例
- [ ] 弹出键盘不遮挡网格（移动端）
- [ ] 桌面端支持物理键盘输入

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端路由 | `shared-backend/src/routes/wordle/` |
| WebSocket | 原生 `ws` `noServer` 模式 |
| 部署 | `cp -r dist/* /root/projects/static/wordle/` |
