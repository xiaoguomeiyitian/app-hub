## 阶段划分

### 阶段一：牌组与规则引擎（P0 核心）

| # | 任务 | 说明 |
|---|------|------|
| T01 | 牌组数据结构 | 144 张牌（万 1-9×4 / 条 1-9×4 / 筒 1-9×4 / 东南西北中发白×4 / 春夏秋冬梅兰竹菊） |
| T02 | 洗牌砌牌 | Fisher-Yates 洗牌，4 方各 18 墩×2（川麻 13 墩） |
| T03 | 摸牌打牌循环 | 庄家 14 张→闲家 13 张，轮 Flow |
| T04 | 吃牌检测 | 上家打出的牌 + 手中 2 张组成顺子 |
| T05 | 碰牌检测 | 任意家打出的牌 + 手中 2 张相同 |
| T06 | 杠牌检测 | 明杠(手中 3 张+别人 1 张) / 暗杠(手中 4 张) / 补杠(已碰+摸到第 4 张) |
| T07 | 优先级系统 | 胡 > 杠 > 碰 > 吃，同级按逆时针顺序 |
| T08 | 基本胡牌判定 | 顺子+刻子+将，递归拆解 |
| F09 | 特殊胡型 | 七对(7 对)、十三幺(13 幺+将)、全不靠、组合龙 |
| T10 | 国标番型 | 81 番逐个实现，从高番到低番 |
| T11 | 单元测试 | 牌型判定、番型计算全覆盖 |

**验收**: 正确判定所有标准胡型，番数计算与国标一致
