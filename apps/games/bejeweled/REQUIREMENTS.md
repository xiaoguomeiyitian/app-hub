# 宝石迷阵 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/bejeweled/`

## 项目概述

经典三消宝石游戏，支持单机多模式 + 在线对战。

## 功能需求

### P0 — 核心功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 网格生成 | 8×8 网格，7 种宝石随机填充 | 无初始匹配 |
| F02 | 交换 | 相邻两颗宝石交换 | 仅交换后有匹配时有效 |
| F03 | 匹配消除 | 3+ 连线消除（横/竖） | 消除正确 |
| F04 | 下落填充 | 消除后上方宝石下落，顶部生成新宝石 | 下落动画流畅 |
| F05 | 连锁反应 | 下落后产生新匹配→继续消除 | 连锁计数正确 |
| F06 | 计分 | 基础分 + 连锁加成 + 连击 Combo | 分数计算正确 |
| F07 | 经典模式 | 无时间限制，无步数限制→无法消除时结束 | 判定正确 |
| F08 | 限时模式 | 60 秒内尽量多得分 | 倒计时+结束 |

### P1 — 联机功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F09 | 1v1 比分 | 同时间比谁得分高 | 实时显示对方分数 |
| F10 | 连击干扰 | Combo 给对方加障碍宝石 | 干扰效果明显 |
| F11 | 好友房 | 房间码 | 2 人同场 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 风格 | 缤纷宝石主题，发光效果 |
| 宝石 | 7 色宝石，各有独特形状/图标 |
| 消除 | 爆炸粒子特效 |
| 连击 | Combo 数字放大+震动 |

## 设备适配

**适配验收清单**：
- [ ] 375/390/412/768/1024/1280/1366/1920 全部无溢出
- [ ] 宝石可拖拽交换（触控）
- [ ] 8×8 网格在最小屏完整可见
