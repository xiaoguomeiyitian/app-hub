# 围棋 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/go-game/`

## 项目概述

围棋游戏，支持 AI 对战 + 在线 1v1 + 教学模式。

## 功能需求

### P0 — 核心功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 棋盘 | 19×19 标准棋盘（支持 9×9/13×13 小棋盘） | 渲染正确 |
| F02 | 落子 | 点击交叉点落子 | 落子正确 |
| F03 | 气的计算 | 计算每块棋的气 | 计算正确 |
| F04 | 提子 | 无气的棋子被提走 | 提子正确 |
| F05 | 打劫规则 | 禁止全局同形（劫争） | 规则正确 |
| F06 | 胜负判定 | 数目法/数子法 | 计算正确 |
| F07 | AI 对战 | 集成 KataGo WASM 或简化 Minimax | AI 有基本实力 |
| F08 | 让子 | 支持让 2-9 子 | 让子放置正确 |

### P1 — 教学功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F09 | 定式教学 | 常见开局定式走法 | 可学习+练习 |
| F10 | 死活题 | 预设死活题挑战 | 可完成 |
| F11 | 落子提示 | AI 提示最佳落点 | 提示合理 |

### P2 — 联机功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F12 | 1v1 匹配 | 支持 9×9/13×13/19×19 匹配 | 15 秒内 |
| F13 | 好友房 | 房间码+棋盘大小选择 | 2 人同局 |
| F14 | 段位系统 | ELO 段位 | 积分正确 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 风格 | 木质棋盘，典雅东方风格 |
| 棋子 | 黑白圆形，径向渐变 |
| 提子 | 提子动画 |
| 目数 | 终局数目显示 |

## 设备适配

**适配验收清单**：
- [ ] 375/390/412/768/1024/1280/1366/1920 全部无溢出
- [ ] 19×19 棋盘在 375px 下可缩放/滚动
- [ ] 9×9 小棋盘在手机端完整可见
- [ ] 交叉点点击区域 ≥30px

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端路由 | `shared-backend/src/routes/go-game/` |
| WebSocket | 原生 `ws` `noServer` 模式 |
| 数据库 | `better-sqlite3`（段位） |
| 部署 | `cp -r dist/* /root/projects/static/go-game/` |
