# 21点（Blackjack）— 需求文档

> 📅 创建日期: 2026-04-02
> 📁 项目目录: `/root/projects/apps/blackjack/`

## 项目概述

经典赌场 21 点纸牌游戏，支持**单人模式（vs AI 庄家）** + **多人竞技联机模式**。

## 功能需求

### P0 — 单人模式

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 发牌 | 玩家 2 张明牌，庄家 1 明 1 暗 | 初始牌面正确 |
| F02 | 要牌（Hit） | 玩家点击要牌，获得新牌 | 超过 21 爆牌判定正确 |
| F03 | 停牌（Stand） | 玩家停止要牌，庄家自动要牌 | 庄家 ≥17 停牌 |
| F04 | 牌值计算 | A=1/11，J/Q/K=10，数字牌面值 | 多 A 自动最优取值 |
| F05 | 胜负判定 | 玩家 vs 庄家，爆牌/点数比较 | 赢/输/平 判定正确 |
| F06 | 筹码系统 | 初始筹码，下注，输赢结算 | 筹码增减正确 |
| F07 | 重新开局 | 一局结束后可继续 | 连续对局流畅 |
| F08 | AI 庄家 | 按标准规则自动要牌（<17 必要，≥17 停） | 行为符合规则 |

### P1 — 联机竞技

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F09 | 快速匹配 | 加入匹配队列，自动凑 3 人 | 8 秒内匹配 |
| F10 | 同步发牌 | 3 人同桌 vs 同一庄家 | 牌面同步 |
| F11 | 轮流操作 | 按顺序轮到操作 | turn 指示正确 |
| F12 | 结算展示 | 一局结束显示所有玩家结果 | 赢/输/平/爆 各自标记 |
| F13 | WebSocket 心跳 | 20 秒 ping/pong | 不断线 |

### P2 — 体验增强

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F14 | 发牌动画 | 牌从牌堆滑到玩家位置 | 平滑过渡 |
| F15 | 翻牌动画 | 庄家暗牌翻转 | 正面显示 |
| F16 | 筹码下注动画 | 筹码堆叠显示 | 视觉反馈 |
| F17 | 音效 | 发牌、翻牌、筹码音效 | 可开关 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 整体风格 | 经典绿绒赌桌，深绿色背景 |
| 牌面 | 标准扑克牌样式，红/黑花色 |
| 筹码 | 彩色圆形筹码（白1/红5/蓝10/黑25/金100） |
| 布局 | 桌面中央庄家，底部玩家，上方牌堆 |

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端 | 已有 `shared-backend/src/routes/blackjack/index.ts` |
| WebSocket | `/api/blackjack/websocket`，noServer 模式 |
| 后端协议 | `match:join` → `match:queued` → `match:found` → `game:deal` → `game:turn` → `game:hit`/`game:stand` → `game:settle` |

## 设备适配

8 分辨率全通过（375/390/412/768/1024/1280/1366/1920），触屏可操作。
