# 掼蛋单机版 REQUIREMENTS

## 需求范围
实现一套可直接游玩的单机掼蛋桌面版。

## 当前已实现功能
- 四家固定座位
- 南家可交互出牌
- AI 自动处理其他三家
- 出牌区固定锚点显示
- 南家手牌连续排序展示
- 计时器、音效、记牌器、烟花结算
- 基础响应式适配

## 交互规则
- 仅南家允许手动操作
- 过牌、出牌、清轮、结算流程可用
- 北家与南家多张牌均横向展示
- 南家手牌保持腾讯掼蛋风格排序

## 验收标准
- 页面可正常进入对局
- 四席不重叠，座位位置稳定
- 南家出牌不压按钮、不压手牌
- 375 / 390 / 412 / 768 / 1024 / 1366 / 1920 分辨率下布局不明显漂移
- 构建可通过
