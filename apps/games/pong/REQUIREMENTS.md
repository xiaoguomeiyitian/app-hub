# 乒乓球 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/pong/`

## 项目概述

经典 Pong 乒乓球，支持 vs AI + 在线 1v1。

## 功能需求

### P0

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 球台 | 横向长方形，中线 | 渲染正确 |
| F02 | 球拍 | 左右各一，上下移动 | 移动正确 |
| F03 | 球 | 来回弹射，碰撞反弹 | 物理正确 |
| F04 | 计分 | 球出界对方得分，11 分制 | 计分正确 |
| F05 | AI | 多级难度（慢/中/快） | AI 追踪球 |
| F06 | 球速递增 | 每次碰撞球速微增 | 递增正确 |

### P1 — 联机

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F07 | 1v1 实时 | WebSocket 同步 | 延迟 < 50ms |
| F08 | 好友房 | 房间码 | 2 人同局 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 风格 | 复古像素，暗色背景 |
| 球 | 拖尾效果 |
| 计分 | 大号像素数字 |

## 设备适配

**适配验收清单**：
- [ ] 375/390/412/768/1024/1280/1366/1920 全部无溢出
- [ ] 移动端手指滑动控制球拍
- [ ] 桌面端鼠标/键盘上下键

| 项目 | 要求 |
|------|------|
