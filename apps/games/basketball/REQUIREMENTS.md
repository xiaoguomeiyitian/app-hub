# 投篮大赛 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/basketball/`

## 项目概述

投篮竞技游戏，支持练习/挑战 + 在线比高分。

## 功能需求

### P0

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 投篮机制 | 触控/鼠标拖拽→角度+力度→抛物线投篮 | 物理合理 |
| F02 | 篮筐 | 固定/移动篮筐 | 碰撞正确 |
| F03 | 计分 | 进球得分，连续进球 Combo 加成 | 计分正确 |
| F04 | 练习模式 | 无限投篮 | 无时间限制 |
| F05 | 挑战赛 | 60 秒限时尽量多进 | 倒计时+结束 |
| F06 | 技巧赛 | 不同角度/距离投篮 | 多位置 |

### P1 — 联机

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F07 | 1v1 比高分 | 同时间比进球数 | 实时显示对方分数 |
| F08 | 好友房 | 房间码 | 同场 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 风格 | 篮球场侧视图 |
| 投篮 | 抛物线轨迹 |
| 进球 | 篮网抖动+庆祝 |

## 设备适配

**适配验收清单**：
- [ ] 375~1920 全部无溢出
- [ ] 触控拖拽投篮流畅
- [ ] 鼠标拖拽可用

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端路由 | `shared-backend/src/routes/basketball/` |
| WebSocket | 原生 `ws` `noServer` 模式 |
| 部署 | `cp -r dist/* /root/projects/static/basketball/` |
