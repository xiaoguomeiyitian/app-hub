# 需求文档 - pinball

> 项目目录: /root/projects/apps/pinball

## 功能需求
- 提供弹球小游戏
- 支持左右方向键控制挡板
- 上方向键发射小球
- 碰撞目标加分，掉落后结束并可重开

## 技术约束
| 项目 | 要求 |
|------|------|
| 前端构建 | Vite + TypeScript (strict) |
| 生产构建 | `base: './'` |
| 开发访问 | `server.host: '0.0.0.0'` |

## 验收标准
- [ ] 页面可正常打开
- [ ] 挡板可左右移动
- [ ] 小球物理与碰撞正常
- [ ] 结束态与重开功能可用
- [ ] TypeScript 与构建通过
