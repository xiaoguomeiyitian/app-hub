# 炸弹人 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/bomberman/`

## 项目概述

经典炸弹人游戏，支持单机闯关 + 在线多人对战。

## 功能需求

### P0 — 核心功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 地图生成 | 网格地图，不可破坏墙壁+可破坏箱子 | 随机生成合理 |
| F02 | 玩家移动 | 上下左右移动，不能穿墙/箱子 | 碰撞正确 |
| F03 | 放置炸弹 | 按键/点击放置炸弹 | 炸弹位置正确 |
| F04 | 爆炸传播 | 炸弹倒计时 3 秒，十字方向传播，被墙阻挡 | 传播正确 |
| F05 | 道具系统 | 加速/增加炸弹数/扩大火焰范围 | 吃到道具生效 |
| F06 | 死亡判定 | 被爆炸波及即死 | 判定正确 |
| F07 | AI 敌人 | 巡逻+追击 AI | 闯关模式有挑战性 |
| F08 | 闯关模式 | 多关卡，消灭所有敌人开门过关 | 关卡递进 |

### P1 — 联机功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F09 | 2-4 人对战 | 同一地图互相炸 | 最后存活者获胜 |
| F10 | 好友房 | 房间码 | 2-4 人同局 |
| F11 | 道具战 | 对战模式下的道具争夺 | 策略性 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 风格 | 像素风，明亮色彩 |
| 地图 | 俯视角网格 |
| 炸弹 | 倒计时闪烁动画 |
| 爆炸 | 火焰扩散特效 |
| 道具 | 闪烁发光图标 |

## 设备适配

**适配验收清单**：
- [ ] 375/390/412/768/1024/1280/1366/1920 全部无溢出
- [ ] 移动端虚拟摇杆+炸弹按钮可用
- [ ] 横屏优先提示（移动端）
- [ ] 桌面端键盘方向键+空格
- [ ] 地图在任何分辨率下完整可见

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |

| WebSocket | 原生 `ws` `noServer` 模式 |
