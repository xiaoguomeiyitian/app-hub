# 谁是卧底 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/spyfall/`

## 项目概述

谁是卧底社交推理游戏，纯联机（6-10 人房间 + AI 补位）。

## 功能需求

### P0

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 房间 | 6-10 人，邀请码 | 加入正确 |
| F02 | 词卡分配 | 多数人拿"卧底词"，1-2 人拿"平民词" | 分配正确 |
| F03 | 描述轮 | 每人说一个词描述自己的词 | 轮流描述 |
| F04 | 投票 | 描述完毕投票淘汰 | 计票正确 |
| F05 | 胜负 | 卧底被投出→平民赢/卧底存活到最后→卧底赢 | 判定正确 |
| F06 | AI 补位 | AI 生成描述词 | 描述合理 |
| F07 | 词库 | 大量词对 | 覆盖面广 |

### P1 — 增强

| # | 功能 | 描述 |
|---|------|------|
| F08 | 白板词 | 1 人什么词都没有 |
| F09 | 自定义词库 | 玩家可上传词对 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 风格 | 简洁卡片式 |
| 词卡 | 翻转显示 |
| 描述 | 气泡对话式 |
| 投票 | 圆形投票圈 |

## 设备适配

**适配验收清单**：
- [ ] 375~1920 全部无溢出
- [ ] 10 人座位可滚动
- [ ] 描述输入框不被键盘遮挡

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端路由 | `shared-backend/src/routes/spyfall/` |
| WebSocket | 原生 `ws` `noServer` 模式 |
| 部署 | `cp -r dist/* /root/projects/static/spyfall/` |
