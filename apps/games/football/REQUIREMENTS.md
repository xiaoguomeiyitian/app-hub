# 像素足球 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/football/`

## 项目概述

像素风足球游戏，支持 vs AI + 在线 1v1。

## 功能需求

### P0

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 球场 | 俯视角像素球场 | 渲染正确 |
| F02 | 球员控制 | 方向移动+传球+射门 | 操控流畅 |
| F03 | 物理 | 球滚动/反弹/弧线 | 物理合理 |
| F04 | AI 对手 | AI 球员跑位+防守 | 有挑战性 |
| F05 | 计分 | 进球计分+时间 | 正确 |
| F06 | 锦标赛 | 多场淘汰赛 | 流程完整 |

### P1 — 联机

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F07 | 1v1 实时 | WebSocket | 延迟 < 100ms |
| F08 | 好友房 | 房间码 | 2 人同局 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 风格 | 像素风俯视角 |
| 球员 | 彩色小人 |
| 进球 | 特效+欢呼 |

## 设备适配

**适配验收清单**：
- [ ] 375~1920 全部无溢出
- [ ] 移动端虚拟摇杆+按钮
- [ ] 横屏优先

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端路由 | `shared-backend/src/routes/football/` |
| WebSocket | 原生 `ws` `noServer` 模式 |
| 部署 | `cp -r dist/* /root/projects/static/football/` |
