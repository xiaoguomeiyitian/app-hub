# 吃豆人竞速 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/pacman/`

## 项目概述

经典吃豆人游戏，支持闯关 + 在线竞速。

## 功能需求

### P0 — 核心功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 迷宫 | 经典吃豆人迷宫，墙壁+通道 | 渲染正确 |
| F02 | 吃豆人移动 | 上下左右控制 | 移动流畅 |
| F03 | 豆子 | 普通豆+能量豆 | 吃到得分 |
| F04 | 幽灵 AI | Blinky(追击)/Pinky(预判)/Inky(协作)/Clyde(随机) | AI 行为符合经典 |
| F05 | 能量豆效果 | 吃能量豆后幽灵变蓝色可吃 | 限时效果 |
| F06 | 生命系统 | 3 条命 | 碰幽灵减命 |
| F07 | 关卡递进 | 吃完所有豆过关 | 关卡递进 |

### P1 — 联机功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F08 | 同图竞速 | 双方看到同一迷宫，比谁先吃完 | 迷宫一致 |
| F09 | 好友房 | 房间码 | 2 人同场 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 风格 | 经典像素风+现代美化 |
| 迷宫 | 蓝色墙壁，黑色通道 |
| 幽灵 | 彩色圆顶造型 |
| 吃豆人 | 黄色圆弧开合动画 |

## 设备适配

**适配验收清单**：
- [ ] 375/390/412/768/1024/1280/1366/1920 全部无溢出
- [ ] 虚拟方向键在手机端可用
- [ ] 迷宫在最小屏完整可见
- [ ] 桌面端方向键/WASD

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端路由 | `shared-backend/src/routes/pacman/` |
| WebSocket | 原生 `ws` `noServer` 模式 |
| 部署 | `cp -r dist/* /root/projects/static/pacman/` |
