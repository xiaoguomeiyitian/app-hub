# 扫雷 — 需求文档

> 📅 创建日期: 2026-04-02
> 📁 项目目录: `/root/projects/apps/minesweeper/`

## 项目概述

经典扫雷游戏（Minesweeper），支持**单人经典模式** + **多人竞技联机模式**。

## 功能需求

### P0 — 单人模式（核心）

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 经典扫雷棋盘 | 格子矩阵，含雷/数字/空白 | 渲染正确 |
| F02 | 左键点击揭示 | 揭示格子，空白自动展开 | 空白区域 flood-fill 展开正确 |
| F03 | 右键插旗/取消 | 标记疑似雷 | 插旗计数正确 |
| F04 | 数字显示 | 1-8 颜色区分（经典配色） | 颜色与经典一致 |
| F05 | 踩雷判定 | 点到雷游戏结束，显示所有雷 | 判定正确 |
| F06 | 胜利判定 | 所有非雷格子揭示即胜 | 判定正确 |
| F07 | 计时器 | 从第一次点击开始计时 | 从 0 开始，精度秒 |
| F08 | 雷计数器 | 剩余未标记雷数 | 插旗/取消实时更新 |
| F09 | 预设难度 | 初级 9×9/10雷、中级 16×16/40雷、高级 30×16/99雷 | 三档切换正确 |
| F10 | 自定义尺寸 | 用户输入宽、高、雷数 | 范围校验（雷数 < 宽×高） |
| F11 | 首次安全 | 第一次点击永远不会踩雷 | 第一点击后重新布雷 |
| F12 | 重新开始 | 笑脸按钮重开 | 重置棋盘、计时、雷计数 |

### P1 — 多人竞技联机

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F13 | 创建房间 | 设定尺寸和雷数，获取房间码 | 房间创建成功 |
| F14 | 加入房间 | 输入房间码加入 | 2-4 人同局 |
| F15 | 自动开局 | 人数 ≥2 自动开始 | 同步揭示 |
| F16 | 竞技揭示 | 多人同时揭示，踩雷淘汰 | 淘汰正确 |
| F17 | 游戏结束 | 最后存活者胜出 | 胜者判定正确 |
| F18 | WebSocket 心跳 | 20 秒 ping/pong | 不断线 |
| F19 | 昵称显示 | 玩家昵称在界面上 | 显示正确 |

### P2 — 体验增强

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F20 | 问号标记 | 右键循环：空→旗→问号→空 | 循环正确 |
| F21 | 双键/中键和弦 | 同时点击揭示周围（已知数字周围旗数匹配时） | 功能正确 |
| F22 | 最佳记录 | 按难度保存最佳时间 | localStorage 持久化 |
| F23 | 音效 | 揭示、插旗、踩雷、胜利音效 | 可开关 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 整体风格 | 经典 Windows 扫雷风格 |
| 格子 | 3D 凸起灰色按钮，按下变平 |
| 数字颜色 | 1蓝 2绿 3红 4深蓝 5暗红 6青 7黑 8灰（经典配色） |
| 笑脸按钮 | 😊 正常 / 😮 揭示中 /😎 胜利 /😵 失败 |
| 计数器 | 红色 LED 数字风格（类似经典七段显示） |
| 棋盘边框 | 3D 凹陷外框 |

## 设备适配

- [ ] 375×667（小手机）无溢出，可操作
- [ ] 390×844（主流手机）无溢出
- [ ] 412×915（安卓）无溢出
- [ ] 768×1024（平板竖屏）无溢出
- [ ] 1024×768（平板横屏）无溢出
- [ ] 1280×720（小屏笔记本）无溢出
- [ ] 1366×768（主流笔记本）无溢出
- [ ] 1920×1080（桌面）无溢出
