# 黑白棋 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/reversi/`

## 项目概述

黑白棋（Othello/Reversi），支持 AI + 在线 1v1。

## 功能需求

### P0

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 棋盘 | 8×8 黑白格 | 渲染正确 |
| F02 | 落子 | 必须夹住对方棋子才能落子 | 规则正确 |
| F03 | 翻转 | 落子后所有被夹住的对方棋子翻转 | 翻转正确 |
| F04 | 胜负 | 棋子多的一方赢 | 判定正确 |
| F05 | AI | Minimax + 位置权重表，多级难度 | AI 有挑战性 |
| F06 | 合法落点提示 | 显示当前可落位置 | 高亮显示 |

### P1 — 联机

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F07 | 1v1 匹配 | 自动匹配 | 10 秒内 |
| F08 | 好友房 | 房间码 | 2 人同局 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 风格 | 绿色棋盘，黑白棋子 |
| 翻转 | 牌面翻转动画 |
| 计数 | 实时显示双方棋子数 |

## 设备适配

**适配验收清单**：
- [ ] 375~1920 全部无溢出
- [ ] 格子 ≥36px 可点击
- [ ] 棋盘完整可见

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端路由 | `shared-backend/src/routes/reversi/` |
| WebSocket | 原生 `ws` `noServer` 模式 |
| 部署 | `cp -r dist/* /root/projects/static/reversi/` |
