# 海战棋 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/battleship/`

## 项目概述

经典海战棋游戏，支持单机 vs AI + 在线 1v1 对战。

## 功能需求

### P0 — 核心功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 布阵 | 10×10 网格，5 艘船（航母5/战列舰4/巡洋舰3/潜艇3/驱逐舰2） | 拖拽放置，不重叠 |
| F02 | 随机布阵 | 一键随机放置所有船 | 布阵合法 |
| F03 | 射击 | 点击对方海域开炮 | 命中/未中判定 |
| F04 | 命中反馈 | 命中：红色标记；未中：白色标记 | 标记正确 |
| F05 | 沉船判定 | 船的所有格被命中→沉船 | 显示沉船轮廓 |
| F06 | 胜负判定 | 全部 5 艘船被击沉→输 | 判定正确 |
| F07 | AI 对战 | 基于概率密度的 AI | AI 有搜索策略 |
| F08 | 回合制 | 双方轮流射击 | 流程正确 |

### P1 — 联机功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F09 | 1v1 对战 | WebSocket 实时对战 | 延迟 < 200ms |
| F10 | 好友房 | 房间码 | 2 人同局 |
| F11 | 布阵阶段同步 | 双方布阵完成后才开始 | 同步正确 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 风格 | 海洋蓝主题，军事风 |
| 网格 | 双 10×10 网格（己方+对方） |
| 船 | SVG 图标，拖拽放置 |
| 命中 | 水花/火焰动画 |
| 沉船 | 沉没下沉动画 |

## 设备适配

**适配验收清单**：
- [ ] 375/390/412/768/1024/1280/1366/1920 全部无溢出
- [ ] 布阵阶段船可拖拽（触控）
- [ ] 双网格在小屏上下排列
- [ ] 方格最小 28×28px

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端路由 | `shared-backend/src/routes/battleship/` |
| WebSocket | 原生 `ws` `noServer` 模式 |
| 部署 | `cp -r dist/* /root/projects/static/battleship/` |
