# game-2048 项目计划

## 项目概述
实现 2048 数字合并游戏，并保留联机扩展能力。

## 开发阶段
1. 单机核心引擎与渲染
2. 联机入口、匹配与消息协议
3. 对手状态与攻击反馈
4. 构建与浏览器验收
