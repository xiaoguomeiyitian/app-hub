# 四子棋 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/connect4/`

## 项目概述

经典四子棋（Connect 4），支持单机 AI + 在线 1v1。

## 功能需求

### P0 — 核心功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 棋盘 | 7 列 × 6 行竖盘 | 渲染正确 |
| F02 | 落子 | 点击列头，棋子落到底部 | 重力下落正确 |
| F03 | 胜负判定 | 四连珠（横/竖/斜）| 判定正确 |
| F04 | 平局 | 棋盘满无四连 | 判定正确 |
| F05 | AI 对战 | Minimax + Alpha-Beta，3 级难度 | AI 有挑战性 |
| F06 | 落子动画 | 棋子从列头滑落到最底部 | 物理动画 |

### P1 — 联机功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F07 | 1v1 匹配 | 自动匹配对手 | 10 秒内 |
| F08 | 好友房 | 房间码 | 2 人同局 |
| F09 | 实时同步 | 落子同步 | 延迟 < 200ms |

## UI 要求

| 项目 | 要求 |
|------|------|
| 风格 | 蓝色竖盘，红/黄双色棋子 |
| 棋子 | 圆形，径向渐变 |
| 落子 | 物理弹跳动画 |
| 胜利 | 连珠高亮+脉动 |

## 设备适配

**适配验收清单**：
- [ ] 375/390/412/768/1024/1280/1366/1920 全部无溢出
- [ ] 列头点击区域 ≥44pt
- [ ] 棋盘在最小屏完整可见

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端路由 | `shared-backend/src/routes/connect4/` |
| WebSocket | 原生 `ws` `noServer` 模式 |
| 部署 | `cp -r dist/* /root/projects/static/connect4/` |
