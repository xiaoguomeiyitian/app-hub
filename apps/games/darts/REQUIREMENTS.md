# 飞镖 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/darts/`

## 项目概述

飞镖游戏，支持练习/减分赛 + 在线对战。

## 功能需求

### P0 — 核心功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 镖盘 | SVG 绘制标准镖盘（20 区+靶心+双倍环+三倍环） | 渲染正确 |
| F02 | 投掷 | 点击/拖拽瞄准+松手投掷 | 落点准确 |
| F03 | 计分 | 根据落点区域计算得分 | 计分正确 |
| F04 | 301/501 减分赛 | 从 301/501 开始减到刚好 0 | 需要双倍结束 |
| F05 | 高分挑战 | 3 轮（9 飞镖）尽量高分 | 计分正确 |

### P1 — 联机功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F06 | 1v1 对战 | 轮流投掷比减分 | 实时同步 |
| F07 | 多人对战 | 2-4 人轮流 | 同步正确 |
| F08 | 好友房 | 房间码 | 同局 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 风格 | 酒吧风格，木质背景 |
| 镖盘 | 圆形 SVG，彩色分区 |
| 飞镖 | 飞行轨迹动画 |
| 计分 | 大字体显示当前/剩余分 |

## 设备适配

**适配验收清单**：
- [ ] 375/390/412/768/1024/1280/1366/1920 全部无溢出
- [ ] 镖盘在最小屏可点击瞄准
- [ ] 触控拖拽投掷流畅
