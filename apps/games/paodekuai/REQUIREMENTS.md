# 跑得快 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/paodekuai/`

## 项目概述

经典跑得快纸牌游戏（先出完为赢），支持单机 AI + 在线多人。

## 功能需求

### P0 — 核心功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 牌组 | 一副 52 张牌（去掉大小王），或 48 张（去掉 2）| 洗牌发牌正确 |
| F02 | 牌型识别 | 单/对/三/三带一/三带二/顺子(5+)/连对(3+)/飞机/炸弹(4张)/连珠 | 识别正确 |
| F03 | 出牌规则 | 必须大过上家或不出，最先出完为赢 | 规则正确 |
| F04 | AI 对战 | 2-3 家 AI | AI 策略合理 |
| F05 | 手牌排序 | 自动排序（按点数+花色） | 排序正确 |
| F06 | 报单/报双 | 最后 1/2 张牌时自动报出 | 其他玩家看到提示 |

### P1 — 联机功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F07 | 匹配 | 3-4 人匹配 | 10 秒内 |
| F08 | 好友房 | 房间码 | 3-4 人同局 |
| F09 | 实时同步 | WebSocket | 延迟 < 200ms |

## UI 要求

| 项目 | 要求 |
|------|------|
| 风格 | 简洁扑克风格 |
| 手牌 | 底部横向排列，滑动浏览 |
| 出牌区 | 中央，飞行动画 |

## 设备适配

**适配验收清单**：
- [ ] 375/390/412/768/1024/1280/1366/1920 全部无溢出
- [ ] 触控 ≥44pt
- [ ] 手牌可滑动
- [ ] 文字 ≥12px

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端路由 | `shared-backend/src/routes/paodekuai/` |
| WebSocket | 原生 `ws` `noServer` 模式 |
| 部署 | `cp -r dist/* /root/projects/static/paodekuai/` |
