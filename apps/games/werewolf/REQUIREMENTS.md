# 狼人杀 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/werewolf/`

## 项目概述

多人社交推理狼人杀游戏，纯联机（6-12 人房间 + AI 补位）。

## 功能需求

### P0 — 核心功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 房间系统 | 创建房间+邀请码，6-12 人 | 加入正确 |
| F02 | 角色分配 | 狼人/预言家/女巫/猎人/守卫/村民 | 分配随机且比例正确 |
| F03 | 夜晚阶段 | 狼人讨论+刀人/预言家查验/女巫毒救/守卫守护 | 阶段流程正确 |
| F04 | 白天阶段 | 讨论+投票放逐 | 投票计数正确 |
| F05 | 胜负判定 | 狼人全灭→好人赢/好人全灭→狼人赢 | 判定正确 |
| F06 | AI 补位 | 缺人时 AI 接管角色 | AI 行为合理 |
| F07 | 角色技能 | 各角色独特技能 | 技能效果正确 |

### P1 — 增强功能

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F08 | 语音聊天 | 白天讨论阶段语音 | WebRTC 或简单实现 |
| F09 | 自定义配置 | 角色数量可配置 | 配置生效 |
| F10 | 战绩统计 | 历史记录 | SQLite |

## UI 要求

| 项目 | 要求 |
|------|------|
| 风格 | 暗色悬疑风 |
| 角色卡 | 精美角色图标 |
| 夜晚/白天 | 背景色切换 |
| 投票 | 气泡动画 |

## 设备适配

**适配验收清单**：
- [ ] 375~1920 全部无溢出
- [ ] 12 人座位在小屏可滚动
- [ ] 投票面板可滑动
- [ ] 触控 ≥44pt

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端路由 | `shared-backend/src/routes/werewolf/` |
| WebSocket | 原生 `ws` `noServer` 模式 |
| 数据库 | `better-sqlite3` |
| 部署 | `cp -r dist/* /root/projects/static/werewolf/` |
