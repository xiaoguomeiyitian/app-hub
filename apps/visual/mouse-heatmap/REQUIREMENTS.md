# 需求文档 - mouse-heatmap

> 创建日期: 2026-03-25
> 项目目录: /root/projects/apps/mouse-heatmap

## 功能需求
- 鼠标热力图展示
- 支持记录鼠标移动轨迹
- 支持实时生成渐变热力图
- 支持导出图片

## 技术约束
| 项目 | 要求 |
|------|------|
| 前端构建 | Vite + TypeScript (strict) |
| 前端端口 | 20009 |
| 生产构建 | `base: './'` |
| 开发访问 | `server.host: '0.0.0.0'` |

## 验收标准
- [ ] 可通过浏览器访问前端页面
- [ ] 热力图轨迹记录正常
- [ ] 图片导出正常
- [ ] TypeScript 编译无错误
