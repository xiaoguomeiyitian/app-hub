# 需求文档 - virtual-aquarium

> 创建日期: 2026-03-25
> 项目目录: /root/projects/apps/virtual-aquarium

## 功能需求
- 基于 Canvas 的水族箱互动页面
- 支持投放鱼种、喂食、投放气泡
- 鱼群具备自由游动、边界避让、追食行为
- 页面包含环境表现：海草、沙底、光束、水面效果
- 支持多鱼种区分展示

## 技术约束
| 项目 | 要求 |
|------|------|
| 前端构建 | Vite + TypeScript (strict) |
| 前端端口 | 20009 |
| 生产构建 | `base: './'` |
| 开发访问 | `server.host: '0.0.0.0'` |

## 验收标准
- [ ] 可通过浏览器访问前端页面
- [ ] 投放鱼种、喂食、气泡功能可用
- [ ] 鱼群会自动游动并追食
- [ ] TypeScript 编译无错误
