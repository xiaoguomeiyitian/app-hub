# 虚拟水族箱

一个基于 Canvas 的前端水族箱应用：可投放不同鱼种、喂食、放气泡，并观察鱼群自由游动与追食行为。

## 功能

- 多鱼种投放：小丑鱼、蓝魔鱼、神仙鱼、热带鱼、灯笼鱼、河豚
- 点击投放鱼、喂食、放气泡
- 鱼会在水族箱中自动游动、躲边、追食
- 气泡、海草、光束、沙底等环境效果

## 启动

```bash
cd /root/projects/apps/virtual-aquarium
npm install
npm run dev
```

## 构建

```bash
npm run build
```
