# 需求文档 - confetti

> 创建日期: 2026-03-25
> 项目目录: /root/projects/apps/confetti

## 功能需求
- 全屏庆祝彩带飘落动画
- 支持纸屑、星星、爆竹粒子混合效果
- 支持鼠标点击触发庆祝动画

## 技术约束
| 项目 | 要求 |
|------|------|
| 前端构建 | Vite + TypeScript (strict) |
| 前端端口 | 20009 |
| 生产构建 | `base: './'` |
| 开发访问 | `server.host: '0.0.0.0'` |

## 验收标准
- [ ] 可通过浏览器访问前端页面
- [ ] 点击后能触发彩带动画
- [ ] 多粒子混合效果正常
- [ ] TypeScript 编译无错误
