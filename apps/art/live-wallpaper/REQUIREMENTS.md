# 需求文档 - live-wallpaper

> 创建日期: 2026-03-25
> 项目目录: /root/projects/apps/live-wallpaper

## 功能需求
- 动态壁纸生成器
- 支持多主题粒子系统（星空/极光/樱花/气泡/矩阵/萤火虫/飘雪）
- 支持鼠标跟随特效
- 支持自定义参数调节

## 技术约束
| 项目 | 要求 |
|------|------|
| 前端构建 | Vite + TypeScript (strict) |
| 前端端口 | 20009 |
| 生产构建 | `base: './'` |
| 开发访问 | `server.host: '0.0.0.0'` |

## 验收标准
- [ ] 可通过浏览器访问前端页面
- [ ] 多主题特效正常
- [ ] 鼠标跟随与参数调节正常
- [ ] TypeScript 编译无错误
