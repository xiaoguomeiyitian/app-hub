# 调色板（Color Palette）— 需求文档

> 📅 创建日期: 2026-04-02
> 📁 项目目录: `/root/projects/apps/color-palette/`

## 项目概述

取色器 + 色板生成 + 色值转换 + 配色方案导出工具。

## 功能需求

### P0 — 核心

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 取色器 | 可视化色轮/色块选择颜色 | 点击选择 |
| F02 | HEX 输入 | 手动输入 HEX 色值 | 实时更新预览 |
| F03 | RGB/HSL 输入 | RGB 和 HSL 值输入 | 三者联动 |
| F04 | 色值转换 | HEX ↔ RGB ↔ HSL 互转 | 转换正确 |
| F05 | 颜色预览 | 大色块预览当前颜色 | 实时更新 |
| F06 | 一键复制 | 点击色值复制到剪贴板 | 复制成功 |
| F07 | 最近颜色 | 最近使用的颜色记录 | ≥10 个 |

### P1 — 增强

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F08 | 配色方案生成 | 根据主色生成互补/类似/三色/分裂互补 | 方案和谐 |
| F09 | 渐变生成 | 两色间生成渐变色阶 | 可选步数 |
| F10 | 对比度检测 | 两色间 WCAG 对比度 | 显示 AA/AAA 达标 |
| F11 | 色板导出 | 导出 CSS/SCSS 变量 | 格式正确 |
| F12 | 拖拽取色 | 上传图片后点击取色 | 取色准确 |

## UI 要求

| 项目 | 要求 |
|------|------|
| 布局 | 左侧取色器，右侧色值信息+配色方案 |
| 色轮 | 圆形色轮 + 明度/饱和度滑块 |
| 色值卡 | HEX/RGB/HSL 各一行+复制按钮 |

## 技术约束

| 项目 | 要求 |
|------|------|
| 前端 | Vite + TypeScript strict + `base: './'` |
| 后端 | 无（纯前端） |
| 部署 | `cp -r dist/* /root/projects/static/color-palette/` |

## 设备适配

8 分辨率全通过。
