# 粒子爱心 - Particle Heart

Canvas 2D 粒子爱心展示应用，包含13种不同风格的心形粒子动画效果。

## 启动

```bash
cd /root/projects/apps/particle-heart
npm install
npm run dev
```

访问：`https://97.383636.xyz/code/20009/particle-heart/`

## 13种爱心样式

| # | 名称 | 说明 |
|---|------|------|
| 01 | 心跳粒子 | 李峋同款经典心形脉冲动画 |
| 02 | 彩虹渐变 | HSL色相循环流光效果 |
| 03 | 鼠标交互 | 跟随鼠标扩散汇聚 + 连线 |
| 04 | 3D旋转 | Y轴旋转透视投影近大远小 |
| 05 | 水晶轮廓 | 半透明闪烁淡蓝星光 |
| 06 | 粒子汇聚 | 随机位置汇聚→心跳→散开 |
| 07 | 火焰粒子 | 底部红色到顶部黄色火焰 |
| 08 | 星空粒子 | 大量闪烁小星+脉冲恒星 |
| 09 | 双爱心交错 | 红+粉双心形反向呼吸 |
| 10 | 音乐节拍 | 节拍脉冲+冲击波+爆发 |
| 11 | 立体水晶 | 3D心形棱线水晶旋转 |
| 12 | 星云漩涡 | 银河旋臂式引力环绕 |
| 13 | 分形递归 | 大心形由小心形组成 |

## 技术栈

- Vite + TypeScript
- Canvas 2D API
- 纯前端，无后端依赖
