# SVG 编辑器 — 需求文档

> 📅 创建日期: 2026-04-01
> 📁 项目目录: `/root/projects/apps/svg-editor/`

## 功能需求

### P0

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F01 | 画布 | SVG 画布，可设置尺寸 | 渲染正确 |
| F02 | 基础图形 | 矩形/圆/椭圆/线/多边形 | 可绘制 |
| F03 | 选择+移动 | 点击选中，拖拽移动 | 操作流畅 |
| F04 | 属性面板 | 填充/描边/透明度/圆角 | 修改即时生效 |
| F05 | 文字 | 添加文字，设置字体/大小/颜色 | 正确 |
| F06 | 图层 | 上移/下移/删除图层 | 层级正确 |
| F07 | 导出 SVG | 导出为 .svg 文件 | 文件正确 |
| F08 | 撤销/重做 | 操作历史 | 可撤销 |

### P1

| # | 功能 | 描述 |
|---|------|------|
| F09 | 路径编辑 | 贝塞尔曲线编辑 |
