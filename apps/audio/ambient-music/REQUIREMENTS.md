# 需求文档 - ambient-music

> 创建日期: 2026-03-25
> 项目目录: /root/projects/apps/ambient-music

## 功能需求
- 生成式氛围音乐播放页，支持自动生成舒缓背景音乐
- 提供至少三种风格：森林 / 海洋 / 星空
- 支持音波可视化反馈
- 支持播放/暂停与基本参数切换

## 技术约束
| 项目 | 要求 |
|------|------|
| 前端构建 | Vite + TypeScript (strict) |
| 前端端口 | 20009 |
| 生产构建 | `base: './'` |
| 开发访问 | `server.host: '0.0.0.0'` |

## 验收标准
- [ ] 可通过浏览器访问前端页面
- [ ] 三种风格可切换
- [ ] 音频生成与播放正常
- [ ] TypeScript 编译无错误
