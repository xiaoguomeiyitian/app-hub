# 需求文档 - virtual-piano

> 创建日期: 2026-03-25
> 项目目录: /root/projects/apps/virtual-piano

## 功能需求
- 虚拟钢琴演奏页，支持键盘与鼠标点击演奏
- 支持至少 4 种音色切换
- 支持音符可视化特效
- 支持基础提示与按键映射说明

## 技术约束
| 项目 | 要求 |
|------|------|
| 前端构建 | Vite + TypeScript (strict) |
| 前端端口 | 20009 |
| 生产构建 | `base: './'` |
| 开发访问 | `server.host: '0.0.0.0'` |

## 验收标准
- [ ] 可通过浏览器访问前端页面
- [ ] 键盘/点击演奏正常
- [ ] 音色切换正常
- [ ] TypeScript 编译无错误
